# Accessibility

This site was built by [Ava Dedio] for DH 101.

I have tried to make it accessible by:

All images include descriptive alt text (description of visuals))
Page headings follow a logical numerical hierarchy (H1 → H2 → H3) and are not used just for visual styling
Navigation links describe their destination clearly — no "click here" or other ambiguous links
The site uses plain Markdown rendered by GitHub Pages, which screen readers handle well
Color contrast in the default GitHub Pages theme meets WCAG AA standards
