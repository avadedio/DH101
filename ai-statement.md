## Accessibility Statement {#accessibility}

Steps I took to make this site usable by different audiences:

- All images include descriptive alt text (e.g. `![A pencil sketch of a face generated by DALL·E](../assets/week03/selfie-1.svg)`)
- Page headings follow a logical hierarchy (H1 → H2 → H3) and are not used just for visual styling
- Navigation links describe their destination clearly — no "click here" links
- The site uses plain Markdown rendered by GitHub Pages, which screen readers handle well

## Sustainability Statement

Steps I took to reduce the environmental footprint of this site:

- **Compressed images** before uploading — use [Squoosh.app](https://squoosh.app) or [TinyPNG](https://tinypng.com) and aim for under 200KB per image
- **No tracking scripts** — this site uses no analytics, social media pixels, or third-party trackers
- **Limited embeds** — where possible I used screenshots or static images rather than live embeds that load external resources on every visit
- **Static site** — GitHub Pages serves static files with no server-side processing, which is significantly lower-energy than a dynamic CMS
